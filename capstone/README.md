# Capstone Project: CryptoTrack Portfolio Dashboard

The grand finale of the AI-Assisted Coding Training.

This demo builds a complete cryptocurrency portfolio dashboard from a client brief using AI -- zero human code.

## Files

- `brief.md` -- The client brief given to the AI
- `expected-output.md` -- What the final result should look like

## How to Run

```bash
# From any empty directory:
claude

# Then paste the capstone prompt from level-08-agent-teams/demos/03-capstone.md
```

## Expected Result

A working React application with:
- MetaMask wallet connection
- ERC-20 token balance detection
- Live USD prices from CoinGecko
- Portfolio distribution pie chart
- 24h price change indicators
- Transaction history
- Responsive dark mode design
- Production build with zero errors
