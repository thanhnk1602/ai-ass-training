# Acme Corp AI Development Plugin

Standard AI toolkit for all Acme Corp engineering projects.

## What's Included

### Skills
- **review** — Code review following Acme coding standards
- **deploy** — Deployment pipeline for staging and production

### Agents
- **security-reviewer** — Automated security audit with OWASP checklist

### Hooks
- **team-rules** — Enforce naming conventions and code standards

### Context
- CLAUDE.md with Acme tech stack, conventions, and patterns

## Installation

```bash
claude plugins add https://github.com/acme/claude-plugin.git
```
